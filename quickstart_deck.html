The Problem

Two bad options.
Neither behaves like a mind.

Option A

Cram it all in

Stuff everything into the context window. Costs scale forever, relevance decays, and the model forgets nothing selectively — it forgets everything equally once it overflows.

Option B

Dump it in a vector store

Flat, undifferentiated recall. No decay, no reinforcement, no consolidation — every memory is permanent and equally weighted, whether it mattered once or matters every day.

A mind doesn't work like either. It strengthens what's reinforced and lets go of what isn't.

The Idea

Two-speed memory,
exactly like a brain.

Hippocampus

Fast · only while attached
  1. Perceive each message
  2. Salience gate — does this matter?
  3. Working memory — small, decaying
  4. Retrieve & inject relevant recall
  5. Consolidate what's reinforced

Forgets what isn't reinforced.

Neocortex

Slow · baked into weights
  1. During "sleep," reinforced memories become training data
  2. A LoRA adapter is fine-tuned via QLoRA
  3. Memories persist in the model itself
  4. Recalls even when MnemonicAi is detached

The model learns you — permanently.

Guardrails

Learns you
without losing itself.

01

Replay buffer

A base-capability replay buffer runs alongside training, protecting against catastrophic forgetting of core skills.

02

Eval-based stopping

Every sleep pass is checked against held-out evals. Training stops early the moment quality starts to slip.

03

Adapter versioning

Every bake is a new adapter version — inspectable, comparable, and instantly rollback-able if something drifts.
